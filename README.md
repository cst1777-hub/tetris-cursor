# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. 게임 화면이 열리면 **시작** 버튼을 눌러 빈 보드를 확인한다.

### VS Code Live Server 사용 (선택)

VS Code의 **Live Server** 확장을 쓰는 경우:

1. `index.html`을 연다.
2. 우클릭 후 **Open with Live Server**를 선택한다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 화면 구조 |
| `style.css` | 화면 스타일 |
| `script.js` | 게임 로직 (현재는 빈 보드 표시만 구현) |

## 현재 상태

- 게임 보드(10×20), 점수, 시작/재시작 버튼, 조작법 안내가 표시됩니다.
- 아직 블록 이동, 회전, 줄 삭제 등의 게임 로직은 구현되지 않았습니다.
