# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. 게임 화면이 열리면 **시작** 버튼을 눌러 플레이한다.

### VS Code Live Server 사용 (선택)

VS Code의 **Live Server** 확장을 쓰는 경우:

1. `index.html`을 연다.
2. 우클릭 후 **Open with Live Server**를 선택한다.

## 조작법

### 키보드 (PC)

| 키 | 동작 |
|----|------|
| `←` (ArrowLeft) | 왼쪽 이동 |
| `→` (ArrowRight) | 오른쪽 이동 |
| `↓` (ArrowDown) | 시계 방향 회전 |
| `↑` (ArrowUp) | 시계 반대 방향 회전 |
| `Space` | 즉시 낙하 (Hard Drop) |

### 터치 (모바일)

화면 하단 버튼으로 조작합니다.

| 버튼 | 동작 |
|------|------|
| `←` `→` | 좌우 이동 |
| `↻` `↺` | 시계 / 반시계 방향 회전 |
| `▼` | 즉시 낙하 |

모바일에서는 보드 크기가 화면에 맞게 자동 조절됩니다.

이동과 회전은 충돌 판정(`canMove`)을 통과할 때만 적용됩니다. 벽이나 고정 블록과 겹치면 회전이 취소됩니다.

## 점수 규칙

한 번에 삭제한 줄 수를 `n`이라 할 때:

**추가 점수 = 10 × n²**

| 삭제 줄 수 | 획득 점수 |
|-----------|----------|
| 1줄 | 10점 |
| 2줄 | 40점 |
| 3줄 | 90점 |
| 4줄 | 160점 |

## 게임 오버 / 재시작

- 새 블록을 스폰할 공간이 없으면 게임 오버입니다.
- **재시작** 버튼을 누르면 보드, 점수, 타이머, 게임 상태가 초기화됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 화면 구조 |
| `style.css` | 화면 스타일 |
| `script.js` | 게임 로직 |

## 현재 상태

- 게임 보드(10×20), 점수, 시작/재시작 버튼, 조작법 안내
- 블록 자동 낙하, 충돌 판정, 고정, 키보드 조작
- 줄 삭제, 점수 계산, 게임 오버, 재시작
- 모바일 반응형 레이아웃 및 터치 조작
